Project Name: Pharmacy Staff Management System

Developed By: GPW Team

Summary: This project was created to facilitate easy management and record keeping of employees working in Pharmacy Concerns. Here Admin has most priveleges and can add the create new users (like manager, pharmacists, salesman), modify records, keep track of sales information thru this sytem. 

Technologies Used: HTML, CSS, Bootstrap, JavaScript for frontend, PHP as a backend and MySQL for the database.

Features:
-->Simple and elegant UI
-->Easy to use
-->Easy to keep and track sales
-->Provides all employee information at one place
-->Inbuilt Hashing machanism for better security
-->Contains all essential functionalities
-->Fully customizable
-->Compact in size
